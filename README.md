##processing
プログラミング作品ポートフォリオ
1. Tully's Coffee ロゴ模写
企業ロゴをProcessingで再現した作品。図形描画とレイヤー制御により、幾何学的なデザインを忠実に再現しました。

技術要素
rect() と for 文による縞模様の生成
ellipse()・noFill()・strokeWeight() による楕円マスキング
描画順序制御によるレイヤー構造の実装
テキストサイズの階層化による視覚的再現


2. 星のカービィ – インタラクティブアニメーション
マウス操作に反応するキャラクターアニメーション。ゲーム的な動作ロジックと視覚効果を実装。

実装機能
マウス追従キャラクター描画
敵キャラクターの自動移動と接触判定
三角関数（sin/cos）による星の幾何学描画
画面端反射、ランダム色変化
frameCount によるアニメーション制御
関数分割によるモジュール設計


4. エレベーターシミュレーション
オブジェクト指向設計を用いたエレベーター挙動シミュレーション。複数オブジェクトの状態管理と相互作用を表現。

設計ポイント
Object 基底クラスから People・Elevator を継承
インスタンス間の状態参照による動作制御
ArrayList による複数人物管理
pushMatrix() / translate() による座標管理
状態フラグによる明確な状態遷移設計
